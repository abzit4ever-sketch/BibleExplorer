# My Bible Explorer

A lightweight, single-page Bible browser built with vanilla JavaScript.
Navigate from Bible version → book → chapter → verse, or search any
version for keywords and references — all client-side, no backend required.

## Features
- 📖 Browse all available Bible translations/versions
- 📚 Drill down through books → chapters → verses
- 🔍 Full-text search within a selected version, with pagination
- 🔗 Shareable hash-based URLs for every view (e.g. `#/verse?version=...&verse=John.3.16`)
- 🎨 Clean, responsive UI styled with Tailwind CSS

## Tech Stack
- **Frontend:** Vanilla JavaScript, HTML, Tailwind CSS (CDN)
- **Data Source:** [scripture.api.bible](https://scripture.api.bible)
- **Routing:** Custom hash-based router (no framework)
